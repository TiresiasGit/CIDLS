システム概要

CIDLSプラットフォームは、コンセプト入力からアプリ生成・ドキュメント一式生成・自律進化保守ループまでを 一気通貫で実現する自律型開発支援基盤である。 個人開発者・小規模チームが商用品質のSW開発成果物を最小工数で生み出し、 日次11:00の自律進化ループで継続的に品質・速度・精度を複利的に向上させる。

制約条件 (Sp3)

技術制約

制約ID種別内容根拠
CON-T01OS Windows 10/11 必須。WSL 使用禁止。 AGENTS.md [T1.1]
CON-T02Python Python 3.11.x。パッケージ管理は uv 専用。pip 使用禁止。 AGENTS.md [T1.1]
CON-T03エンコード スクリプト本文は UTF-8。バッチファイルは Shift-JIS (chcp 932)。Unicode 絵文字禁止。 AGENTS.md [T1.1] [TS.1]
CON-T04devrag index-code コマンド使用禁止 (Python ファイルでゼロベクトルが発生しベクトル検索を破壊)。index (md のみ) を使用すること。 第9回CAPDkA実証 (2026-05-12)
CON-T05devrag config devrag-config.json に code_patterns キーを含めない。公式スキーマ外。 第9回CAPDkA実証 (2026-05-12)
CON-T06devrag search フラグはクエリの前に配置する。例: devrag search --top-k 5 --config $CFG "クエリ" 引数順序バグ対策
CON-T07フロントエンド iframe 使用禁止。SPA + History API で画面遷移する。 AGENTS.md [UI.7]

運用制約

制約ID種別内容
CON-O01Secret 管理 APIキー・Webhook Secret・DB接続文字列を HTML / 配布 ZIP / public 配下に配置しない
CON-O02フォールバック禁止 SimpleFallback・ダミーデータ補完・エラー隠蔽を一切行わない
CON-O03禁止語彙 フォルダ名・変数名に「未分類/その他/不明/TBD/misc/other」を使用しない

前提条件

  • D:\CIDLS が CIDLS リポジトリルートであること
  • D:\production\.venv に Python 3.11 仮想環境が存在すること
  • devrag.exe が %LOCALAPPDATA%\devrag\devrag.exe に存在すること
  • GitHub Copilot が VS Code に設定済みであること
  • AGENTS.md が CIDLS リポジトリルートに存在し、copilot-instructions.md に反映済みであること
